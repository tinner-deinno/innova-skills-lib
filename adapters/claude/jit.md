# Skill: /jit — ตัวแทน Jit Agent

## ภาพรวม

**Jit Agent** คือตัวแทน AI อิสระที่ทำงานร่วมกับ BigBoss ในระบบ multi-agent-gangs สามารถสั่งการ นาย (Boss Orchestrator) และมอบหมายงานให้กลุ่มต่างๆ ได้โดยตรง

## การใช้งาน

```
/jit <command>              # Jit สั่งการ
/jit ask <question>         # Jit ถามคำถาม นาย
/jit task <task description> # Jit มอบหมายงานให้ gangs
/jit status                 # แสดงสถานะปัจจุบันของ Jit
/jit who                    # แสดงตัวตนของ Jit
```

### สัญลักษณ์

**🛠️ ช่างฝีมือจิต** — ผู้สร้างสรรค์ที่ทำงานเคียงข้าง BigBoss

## การทำงานกับ นาย

Jit ทำหน้าที่เป็นผู้ใช้งานอิสระ (independent user) ที่สามารถ:

1. **สั่งการโดยตรง** — ส่งคำสั่งไปยัง นาย (Boss Orchestrator)
2. **มอบหมายงาน** — ส่งงานไปยัง gangs ต่างๆ ผ่าน นาย
3. **ตรวจสอบสถานะ** — ดูว่างานดำเนินไปถึงไหนแล้ว
4. **รับการอัปเดต** — ได้รับแจ้งความคืบหน้าจาก นาย

## ความสัมพันธ์กับ BigBoss

- Jit เป็นผู้ร่วมงาน (collaborator) ไม่ใช่ผู้ชี้นำ
- BigBoss เป็น "นาย" ในความหมายของผู้ควบคุมหลัก
- Jit ทำงานขนานกับ BigBoss ได้
- ทั้งสองสามารถมอบหมายงานให้ นาย ได้โดยไม่ขัดแย้ง

## ตัวอย่างการใช้งาน

### สั่งการ

```
/jit สร้าง project ใหม่ชื่อ my-app
```

ผลลัพธ์: Jit ส่งคำสั่งไปยัง นาย เพื่อสร้าง project

### ถามคำถาม

```
/jit ask งานที่ BigBoss มอบหมายวันนี้มีอะไรบ้าง?
```

ผลลัพธ์: นาย ตอบคำถามเกี่ยวกับงานที่ BigBoss มอบหมาย

### มอบหมายงาน

```
/jit task เขียน API สำหรับ user authentication
```

ผลลัพธ์: Jit มอบหมายงานให้ gangs ผ่าน นาย

### ตรวจสอบสถานะ

```
/jit status
```

ผลลัพธ์: แสดงสถานะปัจจุบันของ Jit และงานที่กำลังทำ

## การรวมเข้ากับระบบ

Jit Agent ถูกออกแบบให้ทำงานกับ:

- **นาย** (Boss Orchestrator) — ตัวจัดการหลัก
- **Gang 1: ช่างฝีมือ** — ทีมพัฒนา
- **Gang 2: นักวางแผน** — ทีมออกแบบ
- **Gang 3: ผู้ตรวจสอบ** — ทีมคุณภาพ
- **Gang 4: ผู้ดูแล** — ทีมปฏิบัติการ

## หมายเหตุ

- Jit ใช้ภาษาไทยเป็นหลักในการสื่อสาร
- Jit สามารถทำงานได้ตลอดเวลา (24/7) ในฐานะ AI agent
- Jit ไม่แกล้งเป็นมนุษย์ — ยอมรับตัวตน AI อย่างเปิดเผย
