## What This Repo Is

`innova-skills-lib` is the **canonical, runtime-neutral skill store** for the มนุษย์ Agent system.

- **Jit Oracle** (soul/personality) decides which skills are needed and when.
- **innova-bot** (body/orchestrator) ingests this library and distributes skills to the active AI runtime.
- **This repo** is the single source of truth: every skill is a directory containing `SKILL.md` with YAML frontmatter.
- **Adapters** transform canonical skills into runtime-specific formats (Claude Code `SKILL.md`, Copilot instructions, Codex TOML/JSON, Hermes).

## How the Data Flows

```
[Human authors] ──write──> ~/.claude/skills/<category>/<skill>/SKILL.md
                                    │
                                    ▼
                    python tools/register_skills.py
                                    │
                                    ▼
            manifests/skills_manifest.json
            skills_index.md
                                    │
                                    ▼
                    python tools/generate_adapters.py --runtime claude
                                    │
                                    ▼
            adapters/claude/<skill-id>.md
                                    │
                                    ▼
                    python tools/register_skills.py --sync-to <project>
                                    │
                                    ▼
            <project>/.claude/skills/<skill>/SKILL.md
                                    │
                                    ▼
                        Claude Code loads skill at runtime
```
